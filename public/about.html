What is SkyReady Aviation?

Here at SkyReady Aviation, we provide a way to access Meteorological Aerodrome Report data (METAR) in a clean and convenient manner. With the site, users have access to over ten fields of data describing attributes of the weather including, temperature, dewpoint, wind direction, wind speed, visibility, altimeter, and sea-level pressure. This data is retrieved from the API provided by the Aviation Weather Center at https://aviationweather.gov/.

Target Browsers

SkyReady Aviation is designed to be accessible by devices of most operating systems including IOS and Android. Browsers hosting the application must be compatible with HTML5, JavaScript, and CSS3, which most are. The incorporated interactive elements are also supported by most environments and device sizes.

Features


Report Fields

Metar ID: A unique identifier assigned to each observation.
Station ID: The ICAO code assigned to the station; A unique four-letter code assigned to each airport by the International Civil Aviation Organization.
Report Time: The timestamp for the METAR report stating the date and time the observation was recorded.
Temperature: Temperature at the location in degrees Celsius.
Dewpoint: The temperature at which the air is completely saturated with water. At a colder temperature dew will begin to form. This dewpoint temperature is useful in assessing the probability of precipitation or cloud formation. This value is measured in degrees Celsius.
Wind Direction: Indicates the direction the wind is blowing in degrees true. This is important for flight planning, for example, runway selection, as wind impacts the way the craft flies.
Wind Speed: The speed the wind is traveling. This also has an effect on the performance of an aircraft, especially in takeoff and landing. The speed is recorded in knots.
Visibility: The distance at which objects can be seen. This data is important as it identifies potential limitations on the pilot's flying ability. This value is recorded in statute miles.
Altimeter: The atmospheric pressure at the station. This value is important for aircraft calibration as it ensures a more accurate altimeter reading during flight. This value is recorded in inches of mercury.
Sea level Pressure: The atmospheric pressure at sea level. This value is used more for comparison purposes between different locations. This value is recorded in millibars.
Coordinates: The geographical coordinates of the station in latitude and longitude format.
